# План внедрения ИИ в UM App

Данный файл содержит идеи и задачи по интеграции искусственного интеллекта на будущее. Будет дополняться по ходу разработки.

## 1. Умный Мэтчинг (Дети ↔ Кружки)
- Связывать результаты диагностики ребенка (интерфейс `Diagnostic` из профиля) с прокачиваемыми навыками курса (поле `skills` в `Course`).
- Выдавать родителю умную подборку: "Этот курс развивает логику на +15%, поэтому он идеально подходит по результатам теста вашего ребенка".

## 2. Персонализированные Дорожные Карты (Roadmaps)
- ИИ должен автоматически разбивать большие цели (например, "Стать программистом") на мелкие еженедельные задачи (`RoadmapItem`).
- Адаптация треков в реальном времени под скорость усвоения материала ребенком.

## 3. Ко-Пилот для Ментора (Учителя)
- Анализ настроения и успеваемости группы.
- ИИ-шпаргалки: советы менторам о том, как лучше подойти к конкретному ребенку на основе его психотипа (выявленного при регистрации/тестах).

## 4. Отчеты для Родителей (Саммаризация)
- ИИ-агрегатор, который собирает оценки, отзывы менторов и посещаемость за наделю, и выдает родителю 1 абзац теплого и понятного отчета о достижениях ребенка.

## 5. Предиктивная аналитика для Организаций
- ИИ-советник, подсказывающий организации: "Курсы по робототехнике сейчас в тренде, у вас есть свободный класс и педагоги, рекомендуем запустить группу".
