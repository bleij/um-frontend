## 🔐 Аутентификация

### Процесс входа
1. Учитель вводит номер телефона на `/login`
2. Система генерирует email: `{phone}@um.app`
3. Функция `signIn()` проверяет учителя в таблице `org_teachers`
4. При успешной авторизации:
   - Сохраняется `teacher_id` в localStorage
   - Создается временный User объект
   - Перенаправление на `/teacher`

### Выход
- Кнопка "Выйти" в профиле
- Очистка localStorage (`teacher_id`, `currentUser`)
- Перенаправление на `/login`

## 👥 API функции (teacher.ts)

1. `getTeacherProfile(userId)` - профиль учителя
2. `getTeacherByPhone(phone)` - поиск по телефону
3. `getTeacherGroups(teacherId)` - группы учителя
4. `getTodayLessons(teacherId)` - уроки на сегодня
5. `getGroupStudents(groupId)` - ученики группы
6. `saveAttendance(groupId, date, attendance)` - сохранить посещаемость
7. `getAttendance(groupId, date)` - получить посещаемость
8. `updateStudentProgress(studentId, skill, value, notes)` - обновить прогресс
9. `getTeacherOrganization(teacherId)` - организация учителя
